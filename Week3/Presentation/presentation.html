Justified Gallery

From the website

This plugin allows users to create images that are high quality, vary in all sizes and fill white space with images.

Download plugin

http://miromannino.com/projects/justified-gallery/

Add Folders

Add plugin folders to your file

JustifiedGallery.css
JustifiedGallery.js

More plugins

Depending on what look and feel you may want for your gallery, you may want to add a few more plugins. For instance in my example I used 'swipebox'.

Start Coding

For a simple Justified Gallery webpage, add

  • CSS Link
    • <link rel="stylesheet" href="css/justifiedGallery.css"></link>
  • JS Script
    • <script src="js/justifiedGallery.js"></script>
  • Your additional Plugin
    • <script src="js/jquery.swipebox.min.js"></script>

HTML tags

Since you will only be working with images be sure to add

  1. <div class="myGallery">
  2. <a href="img/oldfarm_m.jpg" title="What's your destination?">
  3. <img alt="What's your destination?" src="img/oldfarm_t.jpg" />

JS code

Simple Basic Code:

$(".myGallery").justifiedGallery();

JS code

Another type of code to try

Random Size Images

$(".myGallery").justifiedGallery({ randomize : true });
